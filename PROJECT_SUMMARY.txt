🚀 CÓMO EMPEZAR

1. Ejecutar localmente (Python):
   $ cd /Users/cristianbaldini/AlternaTV/alterna-medios
   $ python3 -m http.server 8000
   → Abre: http://localhost:8000

2. Ejecutar localmente (Script):
   $ ./serve.sh
   → Se abre automáticamente en tu navegador

3. Personalizar URLs:
   - Edita index.html
   - Busca: radio.alterna.ar y tv.alterna.ar
   - Actualiza con tus URLs

4. Personalizar colores:
   - Edita style.css (líneas 8-17)
   - Cambia --accent-primary y --accent-secondary

5. Agregar favicon:
   - Copia tu logo a favicon.png (192x192px)
   - Asegúrate que esté en la raíz del proyecto

6. Desplegar online:
   - Vercel: vercel deploy
   - Netlify: netlify deploy
   - GitHub Pages: push a main y habilita Pages

Añadir más plataformas:

  2. Cambia el href y el contenido
  3. Añade estilos en style.css

🎓 INSTRUCCIONES DE USO

Ver el sitio localmente:
  1. Abre Terminal
  2. cd /Users/cristianbaldini/AlternaTV/alterna-medios
  3. python3 -m http.server 8000
  4. Abre http://localhost:8000

Editar contenido:
  - Abre index.html en tu editor
  - Cambia URLs, textos, etc.
  - Guarda (Ctrl+S)
  - Recarga el navegador (F5)

Editar estilos:
  - Abre style.css
  - Personaliza colores, espacios, etc.
  - Guarda y recarga

Desplegar:
  - Vercel: npm install -g vercel && vercel
  - Netlify: npm install -g netlify-cli && netlify deploy
  - GitHub Pages: git push a main

💡 TIPS Y TRUCOS

1. Cambiar rapidez de animaciones:
   app.js: cambiar valores de 300ms, 600ms, etc.

2. Añadir tu logo en header:
   Añadir <img> en el .logo-alterna

3. Cambiar fuente:
   style.css línea 127: font-family

4. Oscurecer fondo:
   style.css línea 7: --bg: #000000;

5. Hacer texto más grande:
   style.css: aumentar clamp() valores

6. Cambiar orden de secciones:
   index.html: mover <section> blocks

7. Añadir formulario de contacto:
   index.html: #contact → añadir <form>

8. Integrar chat:
   Añadir script de Tawk, Drift, etc. antes de </body>
