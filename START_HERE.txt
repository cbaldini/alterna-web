   ✓ Verifica estar en carpeta correcta
   ✓ Intenta puerto diferente: python3 -m http.server 8001
   ✓ Verifica que index.html existe

❌ "No se ven los estilos"
   ✓ Limpia caché: Ctrl+Shift+Delete
   ✓ Verifica que style.css está en la carpeta
   ✓ Abre DevTools (F12) y revisa Network

❌ "Las animaciones son lentas"
   ✓ Edita app.js: cambia 300ms a 500ms
   ✓ O desactívalas en style.css

❌ "Problema en móvil"
   ✓ Abre DevTools → Modo dispositivo
   ✓ Simula iPhone o Android
   ✓ Verifica viewport meta tag

❌ "El favicon no aparece"
   ✓ Asegúrate que favicon.png existe
   ✓ Limpia caché del navegador
   ✓ Intenta refrescar (Ctrl+F5)
